# Rebase Challenge 2022

API em Ruby para listagem de exames médicos.

## Tech Stack

- Docker
- Ruby
- PostgreSQL
- Sidekiq
- Redis

## Running

```bash
$ docker compose up
$ open http://localhost:3000/tests
```
